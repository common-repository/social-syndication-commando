== Installation ==

1. Upload `social-syndication-commando` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add content to posted to the social site through the post add/edit screen
4. Add accounts, username and password will be required for some sites, your password will be encrypted to keep it safe and will not be for any other purposes aside for syndication of content you add to the social sites
